Data Science with Python Pandas¶

Data Scientific Method¶

1. Ask a question - experience, Kaggle, experts

-  What are some issues that you are facing? What would you like to know more about?
-  Data challenges, companies post a question?
-  What are some expert questions which they are looking to answer?
In this case, the questions we can ask about this particular dataset are: 
    1. What are the yearly average land tempatures from 1750 - 2015? 
    2. What temperatue can we predicit in 2050?

2. Get the data - scraping databases, excel/CSVs

3. Explore - pandas, matplotlib, numpy i.e.,

4. Model - pandas, sk-learn - i.e., linear regression, polynomial regression, etc.

5. Communicate - matplotlib, d3.js - once you have that data you can communicate it. Then they can generate actionable goals
